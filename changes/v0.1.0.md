# v0.1.0

First release. A faithful Rust port of Kirk Baucom's Perl asciiquarium (v1.1), animating the full aquarium in your terminal.

## Highlights

- Full cast: fish in both art sets with air bubbles, swaying seaweed, a castle, and a rotating headliner (shark, ship, whale with an animated spout, sea monster, or big fish), plus a `-c` classic mode.
- A hand-rolled compositor stands in for Term::Animation and curses: z-depth painter's algorithm, per-cell transparency, and color masks, all on crossterm.
- Byte-exact art generated from the reference Perl, cell-overlap collisions (shark teeth eat small fish, bubbles pop at the surface), live terminal resize, and a clean terminal restore on exit, panic, and termination signals.
- Runs on Linux, macOS, and Windows.

## Install

- Prebuilt binaries are attached below, so `eget cablehead/asciiquarium-rs` works.
- From source: `cargo install asciiquarium-rs`.
